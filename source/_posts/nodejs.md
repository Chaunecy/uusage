## Install

```bash
sudo apt-get update
sudo apt-get install nodejs
sudo apt install npm
```

## Upgrade

```shell
sudo npm cache clean -f
sudo npm install -g n
sudo n stable
```

## NPM upgrade

```shell
sudo npm i -g npm
```

## Error

### Module not found: Can't resolve 'react-router-dom'

```shell
npm install -S react-router-dom
```
